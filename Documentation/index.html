What is Suppa

Suppa is an excellent responsive, Customizable, Powerfull and User-Friendly WordPress Mega Menu, that allows to create a creative menu with different menu items such as Dropdowns, Mega Posts, Mega Links, Latest Posts, Search Form, Social Media and HTML / Shortcodes !

Suppa is the most customizable menu on the internet . With the ability to customize almost everything from the admin panel area.

How to Install

  1. Upload the plugin through "Plugins > Add New > Upload" interface or upload "suppamenu" folder to the "/wp-content/plugins/" directory
  2. Activate the plugin through the "Plugins" menu in WordPress
  3. Go to Appearence > menus : create your menu items then select theme location and save.
  4. To Style it , go to "Suppa Menu" and start the Fun!

Build a Menu

  1. Go to "Appearence > menus"
  2. Create New Menu
  3. Add Menu Items
  4. Select Menu Location & Skin, then click "Save" ( need to be selected )
  5. Select location also here ( need to be selected )
  6. Finally : Click on "Save Menu" Button

Menu Items Limit

No LIMIT with suppa, the limit is caused by your hosting company, so here's how to fix it.

Fix The Menu Items Limit

1. PHP max_input_vars
Open php.ini and add this line of code

max_input_vars = 3000

if the code is already there , increase the number to 3000 or more ( that depends )
if you don't know what you are doing , contact your host and have them make the change for you.

2. Suhosin
Not all the server have Suhosin ( PHP Security module ) .
Open php.ini and add these lines of code

suhosin.post.max_vars = 3000
suhosin.request.max_vars = 3000

if the code is already there , increase the number to 3000 or more ( that depends )
Again, if you don't know what you are doing , contact your host and have them make the change for you.
